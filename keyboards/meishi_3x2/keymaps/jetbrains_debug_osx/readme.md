# The JetBrains debug(osx) keymap for meishi_3x2

|  index | key1 | key2           | key3       | key4         |
| ------ | ---- | -------------- | ---------- | ------------ |
|  base  | chg  |  alt + ⌘ + R   |        F8  |          F7  |
|  lower |      |        ⌘ + F2  |  alt + F9  |  shift + F8  |

- chg  
  index change(base -> lower)
- alt + ⌘ + R  
  Resume Program
- F8  
  Step Over
- F7  
  Step Into
- ⌘ + F2  
  Stop
- alt + F9  
  Run To Cursor
- shift + F8  
  Step Out
